This project loads its typefaces (Fraunces + Manrope) from Google Fonts via CDN
<link> tags in index.html — no local font files are required to run the site.

This folder is kept as a deliverable per the project spec, and is the place to
drop local/self-hosted font files (e.g. .woff2) later if you want to remove the
Google Fonts CDN dependency for privacy or performance reasons.
